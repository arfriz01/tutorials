February 9, 2015 by Alicia R. Frizzell

What is AngularJS?

AngularJS is a framework for dynamic web applications. It allows you to extend the HTML vocabulary of your application in a way that makes development clear and structured.

"Angular just makes things easier! I love it."

-Anonymous Web Developer

Getting Started

First we need to identify what elements we will be using with Angular. I will be referring to AngularJS as just Angular for the remainder of this blog post. Some key elements with this framework are expressions, directives, controllers and filters. We will touch on a few of these Angular elements in this tutorial.

Directives

We use directives with the prefix ng-

Some good ones to know:

  • The ng-app directive defines an AngularJS application.
  • The ng-model directive binds the value of HTML controls to application data.

The first section of this block of code creates the app directive. This is saved in the app.js file and is referenced in your HTML code.

(function() {
  var app = angular.module('myApp', []);

  app.controller('TutorialController', function(){
    this.tutorials = links;
  });

We will talk about the second portion of this code block in a moment.

As you saw earlier, we use this app directive in our HTML code

Controllers

Controllers are objects that control the data of Angular applications.

Back to that app.js file we were looking at...

(function() {
  var app = angular.module('myApp', []);

  app.controller('TutorialController', function(){
    this.tutorials = links;
  });


The HTML we create that works with our controller:



The name of our controller is TutorialController. We will use it to control the data we have about AngularJS tutorials. In this case, we will be outputting the data in a list. We will use the AngularJS elements to list the name, title, author and an image of various tutorial sites about Angular.
