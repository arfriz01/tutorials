February 9, 2015 by Alicia R. Frizzell

What is AngularJS?

AngularJS is a framework for dynamic web applications. It allows you to extend the HTML vocabulary of your application in a way that makes development clear and structured.

"Angular just makes things easier! I love it."

-Anonymous Web Developer

Getting Started

First we need to identify what elements we will be using with Angular. I will be referring to AngularJS as just Angular for the remainder of this blog post. Some key elements with this framework are expressions, directives, controllers and filters. We will touch on a few of these Angular elements in this tutorial.
